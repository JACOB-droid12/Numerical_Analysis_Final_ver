Module IV

            
Root Finding Workbench

          

          

            Find roots of f(x) = 0 using five methods. Each tab has its own inputs and step-by-step iteration table.
          

        


        
        

          
          
          
          
          
        


        
        

          

            

              
              
              
            

            
Enter f(x) as a one-variable expression in x. Try x^2 - 2, sin(x) - x/2, or e^(-x) - x. Trig functions use the current Angle setting in the sidebar.

            

              
              
              
              
              
              
              
            

            

              Relative tolerance is the default for Bisection because it matches the lecture-style stopping rule.
            

            
Finite-precision default: the selected k-digit chopping or rounding rule drives the displayed iteration path.

            

              Finite-precision sign options (advanced)
              
Only change these if your professor asks bisection decisions to follow chopped or rounded signs.

              

                
                
              

            

          

        


        
        

          

            

              
              
              
            

            
Enter f(x) and its derivative f′(x) separately. The derivative must be entered manually. Trig functions use the current Angle setting in the sidebar.

            

              
              
              
              
              
              
            

          

        


        
        

          

            

              
              
              
            

            
Enter f(x) and two starting points. No derivative needed. Trig functions use the current Angle setting in the sidebar.

            

              
              
              
              
              
              
            

          

        


        
        

          

            

              
              
              
            

            
Like bisection, but uses linear interpolation: c = b − f(b)(b−a)/(f(b)−f(a)). Trig functions use the current Angle setting in the sidebar.

            

              
              
              
              
              
              
            

            

              Finite-precision sign options (advanced)
              

                
                
              

            

          

        


        
        

          

            

              
              
              
            

            
Enter g(x) such that x = g(x) at the root. The iteration is xₙ₊₁ = g(xₙ). Converges when |g′(x)| < 1. Trig functions use the current Angle setting in the sidebar.
