# Roots React Workbench

Standalone React/Vite workspace for the Vercel Roots Workbench.

This package contains everything the React app needs from the legacy numerical
analysis repo: the React UI, Vite/TypeScript config, Vercel metadata, local
engine sources, synced browser runtime engines, and the release/audit scripts.

## Repository Map

| Path | Purpose |
|------|---------|
| `src/` | React Roots Workbench UI |
| `public/legacy/` | Browser-loaded engine files generated by `npm run sync:legacy` |
| `math-engine.js` | Local scalar arithmetic source |
| `calc-engine.js` | Local multi-operation arithmetic source |
| `expression-engine.js` | Local expression parser/evaluator source |
| `root-engine.js` | Local root-finding source |
| `poly-engine.js` | Local polynomial audit dependency |
| `scripts/` | Engine audits, legacy sync, release gate |
| `vercel.json` | Vercel build metadata |
| `docs/WORKSPACE.md` | Agent handoff notes and source boundary |

## Commands

```powershell
npm install
npm run sync:legacy
npm run test:engine
npm run typecheck
npm run build
npm run release:check
```

## Source Of Truth

For React/Vercel work, this workspace is the source of truth. The root engine
files in this workspace feed `public/legacy/`; do not reach into the old legacy
repo unless you are intentionally importing a new upstream engine change.
