# IEEE-754 Module Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a new top-level IEEE-754 teaching module that converts between decimal values and IEEE-754 double-precision bit patterns, explains the result, and ships with lightweight correctness checks.

**Architecture:** Keep the IEEE-754 logic self-contained in its own browser script and expose it on `window`, similar to the existing engines. Implement the feature as a new sidebar tab plus a dedicated panel in `index.html`, styled through focused additions in `styles.css`, and driven by IEEE-specific event/render helpers in `app.js`. Validate the helper first with a small Node VM audit script, then wire the UI around the verified result shape.

**Tech Stack:** Static HTML, vanilla JavaScript, CSS, Node.js, PowerShell, ripgrep, git
