How to Use

Welcome to Spatial Quasar — your browser-based Numerical Methods Calculator.

1. Choose a Module — Use the sidebar (or tab bar on mobile) to switch between Calculator, Error Analysis, and Polynomial Evaluator.

2. Input Values — Enter your parameters in the provided fields. Use decimal notation for precision.

3. Compute — Hit the Compute button to run the selected method. Results appear inline with step-by-step breakdowns.

4. Theme — Toggle between dark and light modes using the switch in the top-right corner.
